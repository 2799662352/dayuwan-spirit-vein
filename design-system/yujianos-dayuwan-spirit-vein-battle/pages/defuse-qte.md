# QTE 破阵页 (Defuse) — 页面级设计覆盖

> **PROJECT:** 与见 OS · 大余湾灵脉争夺战
> **Page:** pages/defuse/index.vue

> Rules here **override** `MASTER.md`. Unlisted rules follow Master.

---

## 布局

- **全屏**, 无导航栏, 无返回按钮 (10 秒后自动结算退出)
- **背景**: `#000000` 纯黑 + 深红色脉动 (敌方阵营色)

## 倒计时 (顶部居中)

- 字体: HUD 风格等宽, `--fs-hero` (64rpx)
- 颜色: 10→5s 金色, 5→0s 红色闪烁
- 辉光: `text-shadow: 0 0 20rpx currentColor`

## 符咒气泡

- 外观: 圆形, 60-90rpx 随机大小
- 颜色: 敌方阵营色 (对手是红则红色气泡, 反之蓝色)
- 内部: 敌方阵营简化符文图案 (SVG)
- 位置: 随机分布, 根据 `DeviceOrientationEvent` 偏移
  - alpha → X 轴偏移 (±30rpx)
  - beta → Y 轴偏移 (±30rpx)
- 出现动画: `scale(0) → scale(1)` + `opacity: 0 → 1`, 200ms
- 被击破: `scale(1) → scale(1.5)` + `opacity: 1 → 0`, 150ms + 碎片粒子
- 未击破超时: 自行消失, `opacity → 0`, 300ms

## 气泡生成节奏

- 前 5 秒: 每 0.8s 生成 1 个
- 后 5 秒: 每 0.5s 生成 1 个 (加速)
- 同屏最多 5 个
- 单个存活: 2s 后自动消失

## 底部状态栏

- 已击破: `X / 10`, 金色数字
- 消耗玄晶: `-2`, 红色小字
- 背景: 半透明黑

## 结算弹窗

### 成功

- 标题: "阵法破解!", 金色辉光
- 奖励: "+15 玄晶" 大字
- 效果: 金色粒子下落 + 胜利音效 + 短震 x3
- 2 秒后自动返回主战场

### 失败

- 标题: "法力不济...", 红色暗调
- 惩罚: "-5 玄晶" 红色
- 效果: 屏幕红色闪烁 + 失败轰鸣 + 长震 x2
- 2 秒后自动返回

## 特殊规则

- 进入此页面前已扣 2 玄晶 (不可退)
- 无暂停/退出按钮 (10 秒强制完成)
- 陀螺仪不可用时: 气泡静态随机位置, 不跟随偏移
